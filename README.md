Cine-Java: Sistema de Reservas de Assentos
Este é um programa de console simples e eficiente desenvolvido em Java para gerenciar a reserva de assentos em uma sala de cinema ou teatro. O sistema permite visualizar o mapa de ocupação em tempo real e realizar reservas de forma interativa.

Funcionalidades
Mapa Interativo: Exibição visual do teatro com índices de linhas e colunas para facilitar a localização.

Gestão de Assentos: Identificação clara de assentos livres [ ] e reservados [X].

Validação de Dados: O sistema impede reservas em posições inexistentes ou em assentos que já foram ocupados.

Interface Amigável: Menu interativo utilizando a estrutura de repetição do-while.

Tecnologias Utilizadas
Linguagem: Java 17+

Paradigma: Orientação a Objetos (POO)

Estrutura de Dados: Matriz bidimensional (Seat[][])

Estrutura do Projeto
O projeto está dividido em pacotes para melhor organização:

entities: Contém as classes de modelo (Theater e Seat).

application: Contém a classe Program com o método main para execução.
